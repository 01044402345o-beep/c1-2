# 📋 IN VOICE 브랜드 광고 스토리보드

**문서 버전**: v1.0
**작성일**: 2025년 7월
**제작 방식**: AI 기반 멀티모달 광고 제작

## 1. 브랜드 아이덴티티

| 항목 | 내용 |
|------|------|
| **브랜드명** | In Voice |
| **타겟** | 60대 여성, 장기요양기관 시설장 |
| **톤앤매너** | 미니멀·따뜻함 (베이지/화이트, 부드러운 자연광) |
| **USP(차별점)** | 음성인식으로 급여제공기록지를 자동 작성 — 타이핑·손글씨 불필요 |
| **광고 목적** | 브랜드 인지 (Awareness) |
| **핵심 메시지** | "말하면 끝, 급여기록이 자동으로 완성됩니다" |

## 2. 캠페인 목표 & 핵심 메시지

```
캠페인 목표:
장기요양기관 시설장(60대 여성)에게 In Voice를 처음 알리고,
"음성으로 기록이 자동 완성된다"는 핵심 기능을 
8초 안에 직관적으로 인지시킨다.

핵심 메시지 (한 문장):
"음성인식으로 급여제공기록지, 이제 말하는 것만으로 완성됩니다."
```

## 4. 프롬프트 수정 전/후 기록 (씬 3 기준)

```
📌 수정 대상: 씬 3 (전) 이미지 생성 프롬프트
```

### 수정 전 프롬프트
```
스마트폰에 대고 말하며 미소 짓는 한국 여성, 사무실 배경, 영화 같은 분위기
```

### 문제점
```
❌ 연령대 미지정 → 20~30대 여성으로 생성됨 (타겟 불일치)
❌ 스마트폰 화면 내용 미지정 → 빈 화면 or 일반 앱 화면으로 생성됨
❌ 배경 톤 미지정 → 차갑고 어두운 사무실로 생성됨
❌ 감정 묘사 부족 → 표정이 어색하고 광고 톤과 불일치
```

### 수정 후 프롬프트
```
스마트폰에 자연스럽게 이야기하는 60대 한국 여성, 
편안하고 미소 띤 표정, 밝고 깔끔한 사무실 배경, 
디지털 돌봄 기록 양식이 작성된 스마트폰 화면, 
부드럽고 따뜻한 조명, 깨끗한 화이트 및 베이지 톤의 인테리어, 
얼굴과 스마트폰을 강조한 클로즈업 샷, 희망적인 분위기, 영화 같은 느낌
```

### 수정 이유 및 결과 변화
| 수정 항목 | 수정 이유 | 결과 변화 |
|----------|----------|----------|
| `60대 여성` 추가 | 타겟(60대 시설장) 일치 | 연령대 맞는 인물 생성 |
| `작성이 완료된 디지털 케어 기록 양식이 표시된 스마트폰 화면` 추가 | 핵심 기능 시각화 필요 | 기록지 화면이 보이는 폰 생성 |
| `미니멀하고 밝은 사무실`, `화이트&베이지` 추가 | 톤앤매너 통일 | 따뜻하고 밝은 배경으로 개선 |
| `희망적인 분위기` 추가 | 전환 씬의 감정 톤 명시 | 희망적·긍정적 표정으로 개선 |
